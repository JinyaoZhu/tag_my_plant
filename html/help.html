1.Download/Update your database

Open "settings" view, under "Database update" input URL of your database.

Input the relative path to root folder of the database which contains the components.xml.

Example:
Assuming your zip file is database.zip and is located at "http://url/of/your/database.zip", it includes folder "data" which includes "components.xml".

Now, the URL of your database is: http://url/of/your/database.zip
the relative path to root folder of the database is: data/.

Click "DOWNLOAD" after you input the correct address. The app will always remember your inputs, so that you don't need to re-enter the address once your want to update your database from the same source.

2.Searching components

With Barcode-scanner: Connect your barcode-scanner(eg: KDC300 from KOAMTAC) via bluetooth with your smart phone, make sure the barcode-scanner is working in HID mode. Scan the barcode, the app will automatically updates the component information and the corresponding datasheet.

Manual search: Your can choose search method in "settings" view, after the search method is changed the app will sort the component list automatically: in ID mode the list will be sorted by A-Z, while in Code model by 0-9. Click to enter manaul search view, enter the ID or Code of the component and click on the item your want to see, the component information will be updated.

In "COMPONENT" view brief infomation of the component will be shown. You can also tap the picture to get a zoom-in view of the component.

4.OPC XML-DA Server

Open "settings" view, under "OPC XML-DA Server" input URL of the server. And then press "SET SERVER", after setting the server address, it is better to restart the app. The connect status will be display behind the "Status" tag in "COMPONENT" view with a small indicator. When the indicator blinking between GREEN and GREY, that means the connection work fine. Otherwise if the indicator turns RED, that means the connection to the server is broken.
